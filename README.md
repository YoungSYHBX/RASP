# RASP Watermarking Framework
main data framework

This repository contains a lightweight version of the core implementation.

**Note:** The codebase is shared with another ongoing unpublished work. Only the code directly related to this paper is released at this stage. Model definitions and datasets are not included. 

## Requirements

- Python 3.8+
- PyTorch 2.1.2
- NumPy
- scikit-learn
- tqdm
- tensorboardX (for logging)
